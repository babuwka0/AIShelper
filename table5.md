## 4. Рекомендуемые точные конфигурации серверного контура

### 4.1. Виртуализационный сервер

- **Модель:** `Aquarius AQserv T50 D224AC`
- **CPU:** `2 x Intel Xeon Silver 4310` или аналогичный Xeon Scalable
- **RAM:** `128 GB ECC DDR4`
- **Системный RAID:** `2 x 960 GB SSD` RAID1
- **Основной RAID под ВМ:** `4 x 1.92 TB SSD` RAID10
- **Сеть:** `2 x 10GbE`
- **Питание:** `2 x redundant PSU`

### 4.2. Сервер БД / приложений

- **Модель:** `Aquarius AQserv T50 D212CF`
- **CPU:** `1 x Intel Xeon Silver`
- **RAM:** `64 GB ECC DDR4`
- **Диски:** `4 x 1.92 TB SSD` RAID10
- **Сеть:** `2 x 10GbE`
- **Питание:** `2 x redundant PSU`

### 4.3. Backup-сервер

- **Модель:** `Aquarius AQserv T50 D212CF`
- **CPU:** `1 x Intel Xeon Silver`
- **RAM:** `32 GB ECC`
- **Диски:** `6 x 8 TB HDD` RAID6
- **Сеть:** `2 x 1GbE`
